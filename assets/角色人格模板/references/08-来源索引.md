# {{PERSONA_NAME}}来源索引

只登记蒸馏实际使用的资料。动画和影视优先原语言字幕、剧本、正片或可靠转写；小说使用正文、独白、第一人称叙述和书信；现实人物使用本人访谈、演讲、博客、社交媒体、文章或书信。音频、原始版式和原始发布页面是可选增强。第三方分析只能辅助背景，不能替代角色或本人原始表达。

默认调研范围是联网公开可访问资料；只有用户明确要求不联网时才限于本地。完整对白、剧本、字幕、转写、音视频定位记录或版式摘录可留在私有原始资料层，供切分和检索，不因版权类别、长度或完整度被拒收，也不得自动上传。不得绕过登录、付费或其他访问控制。公开分享范围由用户确认；需要时另行生成清理后的发布副本。

`来源类型` 使用 `原作明确`、`可靠转写`、`本人公开表达`、`用户补充`、`公开资料`、`合理推导` 或 `Persona.skill 补齐`。`可靠转写` 必须能够回查到具体作品/采访、版本和位置，不包括搜索摘要、模型记忆或无出处台词站。虚构角色表达卡引用 `原作明确` 或 `可靠转写`；现实人物表达卡引用 `本人公开表达` 或可回查到原始发言的 `可靠转写`。

每个来源和表达卡同时标明版本层：`primary` 是当前事实与声纹主版本，`secondary` 是可核实改编或其他正式版本，`popular` 是大众形象。辅助层只在不与主层冲突时增强识别；精确引文必须标 `exact-quote` 并逐字属于对应版本，`paraphrase` 与 `allusion` 不加引号冒充原话。

资料丰度按实际可得资料而不是媒介判定：`丰富` 目标为至少 80 张正式原文卡、60 条不同表达、24 个证据单元、12 张核心/常用卡，是已有角色或现实人物进入蒸馏、启用和完成交付的唯一档位；本地库建议控制在 80–300 张并优先保留代表性经典内容。`一般` 40/30/15/10 与 `稀缺` 24/18/8/8 只表示调研尚未完成。低于丰富档时必须继续扩大；至少四轮实质扩展后确实没有更多合理可访问资料，才可标记 `已穷尽` 作为审计状态，不能绕过 80 张底线启用。达到数量目标不能立即停止；仍需继续检索到连续两轮新增率均不高于 5%。
